### Miscellaneous scripts

#### Installation

```bash
$ ./install.fish misc
$ misc list
$ misc install newsletter
$ newsletter
```

#### Supported scripts

- 📦 misc
- 🌎 knowledge-map (kmap)
- 📦 dropbox (load-to-dropbox)
- 🗞️ newsletter
- 📚 pdf-picker
- 🖥️ vagrant-init (vg-init)
- 🎦 youtube-playlist-sorter (yps)
